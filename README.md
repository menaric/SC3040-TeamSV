# NTU Moons

NTU Moons is a web application developed with the MERN (MongoDB, Express, React, Node.js) stack. It enables students to explore timetable combinations based on constraints, select a preferred option, and modify module timeslots.

## Features
- **Timetable Exploration**: Generate possible timetable combinations based on user-defined constraints.
- **Customizable Options**: Select a preferred timetable and make adjustments to module timeslots.
- **User-Friendly Interface**: Designed for ease of use with an intuitive React-based frontend.
- **Real-Time Data**: Uses MongoDB for efficient data storage and retrieval.
- **Scalable Backend**: Built with Express and Node.js for handling user requests.
